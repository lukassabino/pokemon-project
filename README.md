# Pokemon Project

 Para instalar o projeto realizar o comando abaixo no diretorio do mesmo

### yarn

 Esse comando instalará todas as dependencias do projeto

 Logo após, rodar

### yarn start

O projeto irá rodar no localhost utilizando a porta 3000
